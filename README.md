# Controle de Estoque

Sistema de controle de estoque desenvolvido com **React (frontend)** e **Quarkus (backend)**.

## Tecnologias

* React
* Java + Quarkus
* REST API

## Estrutura

/backend → API
/frontend → Interface do usuário

## Como executar

### Backend

cd backend/inventory-api
./mvnw quarkus:dev

### Frontend

cd frontend
npm install
npm run dev
